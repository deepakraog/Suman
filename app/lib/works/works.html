Online Order Status Tracking Tool

We built this project for internal purpose for CISCO. Where user can track the status of his order online and do modifications if needed through helpdesk but can't change data. There were 3 user roles,
1) Viewer : Can view and export the sheet, But can't make any data changes.
2) Editor : Can edit the row data.
3) Admin : Can Manage the sheet and dependency settings.
This was my first project and I worked as a shadow resource for an UI developer.
Technologies Involved : ExtJS, Java, MySQL, AJAX etc.,
